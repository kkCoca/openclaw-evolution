# OpenClaw DDG Web Search 产品需求

## 背景
### 现状与痛点
- 现有 Brave API 方案需要注册账号并绑定信用卡，不符合免费、免 API、低接入门槛的目标。
- 现有 `DDG Fallback` 已归档，无法继续作为 OpenClaw 内置搜索的长期方案。
- OpenClaw 需要 AI 主动触发的 Web Search 能力，但当前缺少一个稳定、免费、可替换的搜索扩展。

### 期望目标
- 提供一个面向 OpenClaw 内置搜索场景的 Web Search 扩展，主搜索源为 DuckDuckGo，备用搜索源为 Bing 免 API 页面。
- 替换历史 `DDG Fallback` 方案，形成可独立部署、可维护、具备基础反爬能力的新扩展。
- 在常规网络环境下满足以下验收目标：单次搜索响应时间小于 5 秒，返回结果不少于 10 条，整体成功率不低于 95%。

## 功能需求

| 功能 | 验收标准 | 优先级 |
|------|---------|--------|
| 基础搜索 | 输入非空查询词后，系统返回标题、链接、摘要组成的搜索结果列表；默认优先返回 DuckDuckGo 结果 | P0 |
| 相关搜索 | 主搜索源支持时，系统返回相关搜索词列表；主搜索源不支持时允许为空但不得报错 | P0 |
| 多搜索源降级 | DuckDuckGo 请求失败、超时、被限流或解析失败时，系统自动切换到 Bing 免 API 页面搜索 | P0 |
| 可配置输出格式 | 调用方可按场景选择精简结果或详细结果；至少支持 `summary` 与 `detailed` 两种输出视图 | P0 |
| 反爬控制 | 每次请求携带浏览器级请求头，并在请求间加入随机延迟；连续失败时触发短时退避 | P0 |
| 结果数量控制 | 默认返回 10 条结果；允许调用方在安全范围内下调，最大值不得超过 20 条 | P1 |
| 可观测日志 | 每次搜索记录搜索源、耗时、结果条数、失败原因，不记录敏感用户内容以外的额外信息 | P1 |

## 核心流程

### 正常流程
1. OpenClaw 内部搜索模块发起搜索请求，传入查询词、期望结果数和输出格式。
2. Web Search 扩展校验输入参数，生成本次请求上下文。
3. 扩展按反爬策略设置请求头与随机延迟，优先请求 DuckDuckGo。
4. 系统解析 DuckDuckGo 页面结果，提取标题、链接、摘要和相关搜索词。
5. 系统按输出格式组装响应，返回给 OpenClaw 调用方。
6. 系统记录成功日志，包括搜索源、耗时和结果数量。

### 失败处理
| 失败场景 | 处理方式 |
|---------|---------|
| 查询词为空或仅空白字符 | 直接返回参数错误，不发起外部请求 |
| DuckDuckGo 超时 | 记录超时原因并自动切换到 Bing |
| DuckDuckGo 返回限流或验证码页面 | 判定为主源不可用，进入备用搜索源 |
| DuckDuckGo 页面结构变化导致解析失败 | 记录解析失败并切换到 Bing |
| Bing 也失败 | 返回明确失败结果，包含失败类型与可重试标记 |
| 返回结果少于 10 条 | 只返回实际结果，并在元数据中标记结果不足 |
| 相关搜索缺失 | 主结果照常返回，相关搜索字段为空数组 |

## 边界与约束

| 边界 | 说明 |
|------|------|
| 使用场景 | 仅服务于 OpenClaw 内置搜索，不提供公共开放 API |
| 搜索范围 | 仅支持通用 Web Search，不支持图片、视频、新闻、地图等垂类搜索 |
| 输入语言 | 默认支持中英文查询；其他语言不保证相关搜索质量 |
| 输出条数 | 默认 10 条，最大 20 条，避免高频大结果集触发限流 |
| 反爬策略 | 本期仅实现请求头伪装与随机延迟，不引入代理池、浏览器自动化或账号体系 |
| 数据持久化 | 本期不建设长期索引库，不保存搜索结果历史 |
| 兼容关系 | 新扩展发布后，历史 `DDG Fallback` 仅保留归档，不再作为运行依赖 |

## 范围定义
### 范围内
- 面向 OpenClaw 的 Web Search 扩展设计与交付
- DuckDuckGo 主搜索源接入
- Bing 免 API 备用搜索源接入
- 基础搜索结果解析
- 相关搜索词提取
- 输出格式配置
- 基础反爬与失败降级
- 替换历史 `DDG Fallback` 的集成入口

### 范围外
- 代理池、IP 轮换、验证码自动识别
- 多区域搜索策略和个性化排序
- 搜索结果缓存平台或持久化数据仓库
- 图片搜索、新闻搜索、学术搜索等垂类能力
- 面向第三方开发者的公网 API 服务
