```
│                    DESIGNING 阶段完成                        │
│                   (PRD.md + TRD.md 生成)                     │

                              ↓
┌─────────────────────────────────────────────────────────────┐

│  • 从 TRD 提炼技术方案                                        │

                              ↓
┌─────────────────────────────────────────────────────────────┐

│  • 请求用户签字确认                                          │
│  • 提供签字结论选项（通过/条件通过/驳回）                     │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│              用户签字确认                                     │
│  • 聊天窗口回复 / 邮件回复 / 表单填写                         │
│  • 签字结论：✅ 通过 / ⚠️ 条件通过 / ❌ 驳回                   │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│              签字回填（PRD.md）                               │
│  • 更新 PRD.md 第 15 章"用户确认签字"                          │
│  • 填写签字表格（角色/日期/结论/备注）                        │
│  • Git 提交变更                                              │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│              进入下一阶段（roadmapping）                       │
└─────────────────────────────────────────────────────────────┘
```
