# hello-cdf - 产品需求文档 (PRD)

## v1.0.0（2026-04-01）

### 1. 产品概述

hello-cdf 是一个简单的命令行问候工具，用于验证 CDF 研发流程的正确性。

### 2. 功能需求

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 默认问候 | 运行 `hello-cdf` 输出 "Hello, CDF!" | P0 |
| 自定义名字 | 运行 `hello-cdf --name <名字>` 输出 "Hello, <名字>!" | P0 |
| 帮助信息 | 运行 `hello-cdf --help` 显示使用说明 | P0 |
| 版本信息 | 运行 `hello-cdf --version` 显示版本号 | P1 |

### 3. 用户故事

#### 3.1 默认问候
- **作为** 命令行用户
- **我希望** 运行 hello-cdf 时看到默认问候语
- **以便于** 确认工具正常工作

#### 3.2 自定义名字
- **作为** 命令行用户
- **我希望** 通过参数传递名字
- **以便于** 获得个性化的问候

### 4. 验收标准

| 编号 | Given | When | Then |
|------|-------|------|------|
| AC-01 | 用户在终端 | 运行 `hello-cdf` | 输出 "Hello, CDF!" |
| AC-02 | 用户提供名字 | 运行 `hello-cdf --name Alice` | 输出 "Hello, Alice!" |
| AC-03 | 用户需要帮助 | 运行 `hello-cdf --help` | 显示使用说明和参数列表 |
| AC-04 | 用户查询版本 | 运行 `hello-cdf --version` | 显示 "hello-cdf v1.0.0" |

### 5. 非功能需求

- 代码简洁，不超过 50 行
- 启动时间 < 100ms
- 无外部依赖（仅使用 Node.js 标准库）
