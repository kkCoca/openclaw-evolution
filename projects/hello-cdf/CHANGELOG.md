## 版本说明

本项目遵循语义化版本控制（Semantic Versioning）：
- **MAJOR.MINOR.PATCH** (如 1.0.0)
- MAJOR: 不兼容的 API 变更
- MINOR: 向后兼容的功能新增
- PATCH: 向后兼容的问题修复
