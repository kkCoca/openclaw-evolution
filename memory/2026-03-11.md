## 上证报资讯抓取与 AI 热点分析工具

用户请求开发自动化工具，定时抓取上海证券报官网新闻并用 LLM 分析市场热点。

### 项目位置
`~/Projects/Personal/cnstock-analyzer/`

### 已创建文件
- `main.py` - 主入口，支持命令行参数
- `scraper.py` - Playwright 爬虫，抓取滚动新闻
- `analyzer.py` - LLM 分析模块（支持 Gemini/OpenAI/自定义 API）
- `database.py` - SQLite 数据库操作
- `config.py` - 配置文件
- `requirements.txt` - Python 依赖
- `README.md` - 使用文档
- `crontab.example` - cron 定时任务示例
- `install.sh` - 一键安装脚本

### 功能特性
1. **抓取**: 每 4 小时抓取 20 条最新新闻（标题、URL、发布时间）
2. **去重**: SQLite 数据库基于 URL 去重
3. **正文提取**: 可选抓取 `.content` 类名下正文
4. **AI 分析**: 
   - 关键词提取和聚类
   - 对比过去 3 天数据识别"急剧升温"关键词
   - 生成 Markdown 格式《上证报每日热点内参》

### LLM 配置
支持三种 API：
- Gemini 1.5 Flash（`GEMINI_API_KEY`）
- OpenAI GPT-4o-mini（`OPENAI_API_KEY`）
- 自定义 OpenAI 兼容 API（如硅基流动）

### 下一步
用户需要：
1. 运行 `./install.sh` 安装依赖
2. 配置 LLM API Key
3. 测试运行 `python main.py`
4. 配置 cron 定时任务（可选）

### 注意事项
- 需要安装 Playwright 浏览器：`playwright install chromium`
- 遵守网站 robots.txt 协议
- LLM 调用会产生 API 费用
