用户决定采用 XDG Base Directory Specification 作为工作学习电脑的目录规范。


### 环境变量配置

已配置到 `~/.bashrc` / `~/.zshrc`：

```bash
# XDG 基础目录
export XDG_CONFIG_HOME="$HOME/.config"
export XDG_DATA_HOME="$HOME/.local/share"
export XDG_CACHE_HOME="$HOME/.cache"
export XDG_STATE_HOME="$HOME/.local/state"
export XDG_BIN_HOME="$HOME/.local/bin"

# XDG 用户目录
export XDG_DOCUMENTS_DIR="$HOME/Documents"
export XDG_DOWNLOAD_DIR="$HOME/Downloads"
export XDG_DESKTOP_DIR="$HOME/Desktop"
export XDG_MUSIC_DIR="$HOME/Music"
export XDG_PICTURES_DIR="$HOME/Pictures"
export XDG_VIDEOS_DIR="$HOME/Videos"
export XDG_PUBLICSHARE_DIR="$HOME/Public"
export XDG_TEMPLATES_DIR="$HOME/Templates"

# PATH
export PATH="$HOME/.local/bin:$HOME/.npm-global/bin:$PATH"

# 常用别名
alias projects='cd ~/Projects'
alias work='cd ~/Projects/Work'
alias personal='cd ~/Projects/Personal'
alias learning='cd ~/Learning'
alias config='cd ~/.config'
```

### 一键创建命令

```bash
# XDG 基础目录
mkdir -p ~/.local/{bin,share,state} ~/.config ~/.cache

# XDG 用户目录
mkdir -p ~/Documents ~/Downloads ~/Desktop ~/Music ~/Pictures ~/Videos ~/Public ~/Templates

# 开发者补充目录
mkdir -p ~/Projects/{Work,Personal,Archive} ~/Learning/{Courses,Books,Notes,Practice}
```

### 备份策略

| 目录 | 备份 | 说明 |
|------|------|------|
| `~/.config/` | ✅ | 重要配置 |
| `~/.local/share/` | ✅ | 应用数据 |
| `~/.local/bin/` | ✅ | 个人脚本 |
| `~/.cache/` | ❌ | 缓存可重建 |
| `~/Documents/` | ✅ | 重要文档 |
| `~/Downloads/` | ❌ | 临时文件 |
| `~/Projects/` | ✅ | Git 仓库另算 |
| `~/Learning/` | ✅ | 学习资料 |
| `~/Desktop/` | ❌ | 临时放置 |
