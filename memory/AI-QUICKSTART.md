## 📚 核心概念 (5 分钟理解)

### LLM (大语言模型)
```
就像一个读过很多书的超级大脑
- 能理解你的话
- 能生成回复
- 但不完美，会犯错
```

### Prompt (提示词)
```
就是你对我说的话
- 越清晰，我理解越准确
- 给上下文，我回答更好
- 可以让我扮演角色
```

### Token
```
AI 的"字数单位"
- 输入 + 输出 = 总 Token
- 关系到成本和速度
- 需要优化
```

### Agent (智能体)
```
LLM + 工具 + 记忆 = Agent
- LLM 是大脑
- 工具是手脚
- 记忆是经验
```

### Context Window (上下文窗口)
```
我能记住的最大对话长度
- 越大能记住越多
- 但成本也越高
- 需要合理管理
```
