## 🎯 核心使命

为 OpenClaw 打造**零配置、免 API Key**的扩展能力。

| 维度 | 传统方式 | Universe 方案 |
|------|---------|--------------|
| **使用门槛** | 需注册平台、配置 API Key | 零配置，开箱即用 |
| **成本** | $5-30/月 | 完全免费 |
| **隐私保护** | 商业服务可能有日志 | 隐私优先，本地处理 |
